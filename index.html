OneClick applies a curated set of Windows tweaks — disabling telemetry, trimming background services, and tuning system defaults. Free, no install required.

Before you run it: OneClick makes broad system changes. Some tweaks may affect software that depends on Windows services being enabled. A restore point and registry backup are created automatically — but read the script before running it.

Telemetry & Diagnostics

Background Services

Disables 150+ non-essential Windows services including SysMain, Xbox, Bluetooth, print spooler, and manufacturer-specific services (Intel, AMD, NVIDIA, Razer, Logitech, HP).

Privacy & Bloat Removal

Removes 55+ Microsoft store apps, Edge, OneDrive, Xbox bloat (22 files), Smartscreen, LockApp, and sync programs. Disables advertising ID, location, and content delivery.

Visual & System Settings

Disables animations, transparency, Aero peek, taskbar search, widgets, and news. Sets taskbar left-aligned, enables right-click Win10 menu, applies performance-tuned visual settings.

Power, Kernel & Latency

How it works

No magic.

It's a script. Here's exactly what happens when you run it.

01

Checks version & downloads tools

Compares against the latest version on GitHub. Downloads the OneClick Tools folder (NSudo, Timer Resolution, Nvidia Profile Inspector, OOShutup10, power plans, etc.) before proceeding.

02

Creates restore point & registry backup

Offers a System Restore snapshot and a full registry export (HKLM, HKCU, HKCR, HKU, HKCC) to your OneClick Backup folder. Roll back any time from Windows Recovery or regedit.

03

Applies system, privacy & visual tweaks

Registry keys are written to disable telemetry, configure visual settings, and adjust service startup types. Removes bloat apps, Edge, OneDrive, Xbox files, and Smartscreen.

04

Disables services & drivers

150+ non-essential Windows services and 70+ unnecessary kernel drivers are disabled. Manufacturer services (Intel, AMD, NVIDIA, Razer) are also targeted. Re-enable any from services.msc.

05

GPU-specific tweaks

Auto-detects your GPU vendor and applies driver registry tweaks. Nvidia users get Profile Inspector config. AMD users get clock/power gating disabled. Both paths are skippable.

06

Latency, power plan & timer

BCDEdit tweaks applied (no platform clock, no dynamic tick). Custom Ultimate Performance power plan imported. Priority separation, kernel timer policy, and SetTimerResolution configured.

07

Cleanup & restart

Temp files, prefetch, Discord/Spotify cache, and Windows Update logs cleared. Network stack flushed. Extras menu offered before final restart.

Requirements

Admin rights

Run as Administrator. The script modifies system-level registry keys, service states, and driver startup values.

Windows 11 (21H2–26H1)

Officially supported range. Unsupported versions can still continue with a warning. Windows Server editions are also handled.

Internet connection required

Tools, Dcontrol, OpenShell, and the Nvidia Control Panel are downloaded at runtime from GitHub. Retry and manual fallbacks are provided if downloads fail.

Some things may break

Wifi, Xbox Game Bar, Windows Search, and print features are affected. Process Destroyer (optional) requires a local account — running it on a Microsoft account causes a black screen on login.
